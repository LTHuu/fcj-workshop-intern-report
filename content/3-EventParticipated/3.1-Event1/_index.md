# Report: “AWS First Cloud AI Journey Community Day 2026”

### Purpose

- Connect with cloud and tech experts  
- Explore Cloud & Generative AI  
- Experience real-world demos  
- Expand networking opportunities  

### Speakers

## Key Topics

### GoTyme & GoTymeX

- Digital banking in emerging markets (South Africa, Philippines, etc.)  
- GoTymeX: technology hub in Vietnam  
- ~70% of engineers are Vietnamese  
- Focus on SME support and cost optimization  
- Internship programs (3–6 months) with dedicated mentors  

### Platform Engineering

- Package services for reuse  
- Enable developers to self-deploy  
- Focus on speed, scalability, and reliability  

### GenAIOps & DevOps

- Shift from monolithic to microservices  
- CI/CD + IaC + observability  
- Pipeline: Plan → Code → Build → Deploy → Monitor  

**Key highlights:**
- Generative AI introduces new complexity  
- Emergence of GenAIOps pipelines  
- Large-scale agents (Amazon Bedrock)  

**Tools:**
- Langfuse (monitoring)  
- OpenTelemetry (observability standard)  

### Agentic Development

**Traditional:** one developer, one task  
**Modern:** multiple agents working in parallel  

- Git worktree for isolation  
- Agent supervisors for orchestration  
- Developers act as supervisors  

**Benefits:**
- Parallel execution → faster delivery  
- Automated commits and bug fixing  

### Multimodal GenAI

- Unified embeddings for text, images, video  
- GraphRAG for enterprise knowledge  
- Multi-agent workflows  
- Built-in guardrails and monitoring  

### FCAJ Community

- Inclusive environment, including hearing-impaired members  
- Strong support for newcomers  
- Inspiring perseverance and passion
