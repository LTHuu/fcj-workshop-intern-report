- Kết nối chuyên gia Cloud & công nghệ  
- Khám phá Cloud & Generative AI  
- Trải nghiệm demo thực tế  
- Mở rộng networking

## Nội dung chính

### GoTyme & GoTymeX

- Ngân hàng số tại thị trường mới nổi (Nam Phi, Philippines…)  
- GoTymeX: trung tâm công nghệ tại Việt Nam  
- 70% kỹ sư là người Việt  
- Hỗ trợ SME, tối ưu chi phí  
- Có chương trình thực tập 3–6 tháng, mentor riêng  

### Platform Engineering

- Đóng gói dịch vụ để tái sử dụng  
- Dev có thể tự triển khai, giảm phụ thuộc vận hành  
- Mục tiêu: nhanh, ổn định, dễ mở rộng  

### GenAIOps & DevOps

- Chuyển từ monolithic → microservices  
- CI/CD + IaC + observability  
- Pipeline: Plan → Code → Build → Deploy → Monitor  

**Điểm mới:**
- AI tạo thêm độ phức tạp  
- Xuất hiện GenAIOps pipeline  
- Agent chạy ở quy mô lớn (Amazon Bedrock)  

**Công cụ:**
- Langfuse (monitoring)  
- OpenTelemetry (chuẩn observability)  

### Agentic Development

**Cũ:** 1 người – 1 task  
**Mới:** nhiều agent chạy song song  

- Git worktree để tránh conflict  
- Agent supervisor điều phối  
- Dev trở thành người giám sát  

**Ưu điểm:**
- Làm song song → nhanh hơn  
- Tự động commit, fix bug  

### GenAI đa phương tiện

- Embedding chung cho text, ảnh, video…  
- GraphRAG cho dữ liệu doanh nghiệp  
- Multi-agent workflow  
- Có guardrails + monitoring  

### Cộng đồng FCAJ

- Thành viên khiếm thính vẫn phát triển mạnh  
- Chủ động hỗ trợ người mới  
- Truyền cảm hứng về sự kiên trì
